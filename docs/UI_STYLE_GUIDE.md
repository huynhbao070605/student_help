# UI Style Guide

## Direction

Student Help should feel cute, warm, pastel, Vietnamese, student-friendly, and mobile-first. It should feel like a real mobile app designed for daily student life around VNU-HCM, Thu Duc, and Di An.

Avoid a corporate SaaS feeling. Avoid desktop-style layouts squeezed into mobile.

## Visual Personality

- Friendly and practical.
- Soft pastel colors with warm accents.
- Vietnamese student vibe through copy, examples, content, and small details.
- Clear trust and safety cues.
- Comfortable spacing and large touch targets.

## Layout

- Use bottom tabs for primary student navigation.
- Use cards for posts, vendors, items, alerts, and services.
- Use sheets or modals for filters, quick actions, and confirmations.
- Use sticky bottom actions on detail screens.
- Avoid dense tables.
- Avoid tiny controls.
- Keep list screens filterable and scannable.

## Components

- Rounded buttons.
- Soft shadows.
- Friendly empty states with useful actions.
- Badges and chips for status, category, reputation, and verification.
- Avatar and image-forward cards.
- Thumb-friendly forms.
- Clear loading, error, pending, approved, rejected, and blocked states.

## Copy

- Vietnamese-first where appropriate for demo.
- Short, friendly, and clear.
- Avoid stiff admin language in student flows.
- Safety and privacy text must be direct and easy to understand.

## Motion

- Smooth but lightweight.
- Use subtle transitions for tab changes, cards, sheets, and status updates.
- Avoid heavy animations that hurt Android performance.
